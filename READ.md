# Hoot

Hoot is a Yelp like web app that allows users to view and write reviews for restaurants and local venues.  

## Technologies:

JavaScript, MongoDB, Google Maps API, Passport.js

## Data Models, Routes, Views

### Models

	- User
	- Review
	- Restaurant
	- Follow

### Routes
	
	- Sign up
	- Login
	- Logout
	- Profiles
	- Restaurant list
	- Restaurant
	- Add Restaurant
	- Add Review

### Views

	- Sign up
	- Login
	- Single Profile
	- Single Restaurant
	- All Restaurants
	- All Profiles
	- New Restaurant
	- New Review
